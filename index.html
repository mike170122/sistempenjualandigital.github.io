Laporan & Export

🔄 Sinkronisasi Data Multi-Device

Bagikan data penjualan antar HP/Laptop dengan mudah:

📤 Cara Export (Pengirim)
  1. Klik tombol "Export/Excel" di atas
  2. Mobile: Pilih Download/Share/Copy
  3. Desktop: File auto-download
  4. Kirim file via WhatsApp/Email
📱 Tips Mobile: Jika JSON gagal, gunakan Excel export yang lebih kompatibel
📥 Cara Import (Penerima)
  1. Terima file dari pengirim
  2. Klik "Import JSON" atau "Import Excel"
  3. Pilih file yang diterima
  4. Data otomatis tersinkronisasi
📊 Import Excel (Alternatif 1)
  1. Mobile: Export ke Excel
  2. Kirim file .xlsx via WhatsApp
  3. Admin PC: Import Excel
  4. Data otomatis dikonversi
💡 Keunggulan: Excel lebih kompatibel di semua device & mudah diedit manual
📄 Import PDF (Alternatif 2)
  1. Mobile: Export ke PDF
  2. Kirim file .pdf via WhatsApp
  3. Admin PC: Import PDF
  4. OCR otomatis ekstrak data
🔍 Keunggulan: PDF universal, OCR smart parsing, backup visual data

✅ Keamanan: Data tidak duplikat, merge otomatis, tracking lengkap. Mendukung JSON, Excel & PDF import.
